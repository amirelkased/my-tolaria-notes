# Communication Protocols

**🌐 Popular Modern Protocols**

- **WebSockets**: A "living standard" that creates a persistent, full-duplex connection over a single TCP channel. It is the go-to for chat apps, gaming, and live financial dashboards because it avoids the overhead of constant polling.
- **gRPC**: Built on HTTP/2, it supports "bidirectional streaming" where both client and server send a sequence of messages independently as they are generated.
- **HTTP/2**: Unlike the original request-response HTTP, version 2 supports multiplexing, allowing multiple streams of data to flow back and forth at once.
- **WebRTC**: Enables direct peer-to-peer (P2P) bidirectional communication for video calls and file sharing without necessarily going through a central server.
